1) Import/add files "Flash.ini" and "SAM7.s" of this folder in keil project as 'text file'!

2) For load of *.afx project file into flash use the following syntax:

	a) if "FLASH.ini" is in an upper directory:

		"..target_dir\\dir2\\dir1\\project_name.afx"

	b) if "FLASH.ini" is on the same dir level:

		"DIR\\project_name.afx"
